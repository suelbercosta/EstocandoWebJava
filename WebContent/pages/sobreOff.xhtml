O sistema ESTOCANDO WEB JAVA foi desenvolvido com o intuito de ser o trabalho de conclusão de curso (TCC) do curso de Sistemas de Informação da Faculdade Estácio do Recife. Este sistema foi desenvolvido pelo aluno SUELBER COSTA DA PAZ em computador pessoal utilizando a liguagem de programação JAVA para o Backend, as linguagens XHTML, CSS, JavaScript para o Frontend e a linguagem SQL para o Banco de Dados relacional. O sistema ESTOCANDO WEB JAVA é uma aplicação web desenvolvida em Java e terá como principal função o controle de estoque (entrada e saída) de materiais em um almoxarifado. O sistema não contará com interação cliente/fornecedor, visto que será de uso restrito a uma empresa, semelhante a uma ERP.
